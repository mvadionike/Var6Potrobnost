# Var6Potrobnost
# Определение потребности предприятия в продукции растениеводства

## Описание проекта

Данное приложение разработано на Python с использованием библиотеки PyQt5.
Программа предназначена для расчёта потребности сельскохозяйственного предприятия в продукции растениеводства.

Приложение позволяет:

* рассчитывать плату за паи;
* определять потребность животноводства в кормах;
* учитывать договорные обязательства;
* вычислять итоговую потребность предприятия;
* отображать результаты в виде диаграммы.

---

## Используемые технологии

* Python 3
* PyQt5
* Matplotlib

---

## Функциональные возможности

### 1. Выбор типа продукции

Поддерживаются:

* ячмень;
* пшеница;
* подсолнечник.

Для каждого типа автоматически изменяются:

* размер арендной платы;
* договорная потребность.

---

### 2. Расчёт платы за паи

Формула:

Плата за паи = Численность работников × Размер арендной платы

---

### 3. Расчёт потребности животноводства

Формула:

Потребность животноводства = Количество животных × Норматив расхода корма

---

### 4. Итоговая потребность

Формула:

Итоговая потребность =
Плата за паи +
Потребность животноводства +
Договорные обязательства

---

### 5. Построение диаграммы

Программа строит круговую диаграмму распределения потребностей:

* паи;
* животноводство;
* договорные обязательства.
