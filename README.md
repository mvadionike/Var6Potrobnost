# Var6Potrobnost
# Определение потребности предприятия в продукции растениеводства

## Описание проекта

Данное приложение разработано на Python с использованием библиотеки PyQt5.
Программа предназначена для расчёта потребности сельскохозяйственного предприятия в продукции растениеводства.

Приложение позволяет:

* рассчитывать плату за паи;
* определять потребность животноводства в кормах;
* учитывать договорные обязательства;
* вычислять итоговую потребность предприятия;
* отображать результаты в виде диаграммы.

---

## Используемые технологии

* Python 3
* PyQt5
* Matplotlib
